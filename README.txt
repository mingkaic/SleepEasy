SleepEasy Program Version 1.0 27/10/2014

— GENERAL USAGE NOTES
————————————————————
- When first downloading the application, registration screen will appear, but will offer the option to login without registering

- By default the user will be able to record their sleep time and set the alarm clock.

- The user can visit any screen through the tab on the bottom of the screen

- The user can log out of the application at any time by visiting the profile screen

- The user must enable location to use the exercise recording feature
